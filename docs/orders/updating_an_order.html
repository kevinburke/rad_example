Orders API

Updating an order

PUT /orders/:id

Parameters

Name Description
name Name of order
paid If the order has been paid for
email Email of user that placed the order

Request

Headers

Accept: application/json
Content-Type: application/json
Host: example.org
Cookie:

Route

PUT /orders/1

Body

cURL

curl "http://rad-example.herokuapp.com/orders/1" -d "{"order":{"name":"Updated Name"}}" -X PUT -H "Accept: application/json" -H "Content-Type: application/json" -H "Host: example.org" -H "Cookie: "
